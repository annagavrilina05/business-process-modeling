# Моделирование бизнес-процессов

В данном репозитории представлены некоторые практические работы, которые были выполнены в рамках дисциплины «Моделирование бизнес-процессов».

В ходе изучения курса были освоены различные нотации и инструменты моделирования:

*   **IDEF0**
*   **DFD** 
*   **BPMN** 
*   **eEPC** 
*   **Дерево функций** 
*   **Организационная модель** 

## Инструменты
*   [Ramus](https://ramussoftware.com/) — для IDEF0 и DFD.
*   [Stormbpmn](https://www.stormbpmn.com/) — для диаграмм BPMN.
*   [Sila Union](https://silaunion.ru/) — для eEPC, деревьев функций и организационных моделей.
